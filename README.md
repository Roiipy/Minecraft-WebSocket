# Minecraft-WebSocket
Pythonでマインクラフトと通信します。
[紹介動画](https://youtu.be/84mRasekjC8)

# 推奨設定
Minecraft Windows Edition (Education Edition) 1.17 以上
Python 3系(3.8.2で動作確認済み）

# 必要なモジュール
・asyncio
・websockets
・json
・uuid (uuid4)
